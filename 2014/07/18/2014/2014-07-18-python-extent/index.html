Python是一门极其灵活和简洁的语言，但其执行效率并不理想，相反C/C++的写法相对来说就复杂多了，但是C/C++却有很好的执行效率。而Python提供了可扩展的底层API给编程人员，可以通过它编程实现功能后，给Python上层调用，这样就将Python的灵活性和C/C++的效率结合起来了~
